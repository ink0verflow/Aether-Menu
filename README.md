# Aether

**Aether** is a mod menu for splatoon 1 on aroma

## Required

- **Console:** Nintendo Wii U  
- **Environment:** Aroma
- **Game:** Splatoon 1  

## Controls

| Input | Action |
|-------|--------|
| **L + R + MINUS** | Open / close menu |
| **D-Pad** | Navigate |
| **A** | Toggle cheat |
| **B** | Back / close |
| **Y** | Show / hide active cheats list |
