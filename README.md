# Aether

**Aether** is an in-game mod menu plugin for **Splatoon 1** on **Wii U**, built for the **Aroma** environment (WUPS).

Overlay menu with cheat management, category navigation, active-cheat HUD, and persistent save states. Designed for personal / private use.

## Platform

- **Console:** Nintendo Wii U  
- **Environment:** Aroma (WUPS)  
- **Game:** Splatoon 1  
- **Required modules:** KernelModule (memory writes), NotificationModule (recommended)

## Controls

| Input | Action |
|-------|--------|
| **L + R + MINUS** | Open / close menu |
| **D-Pad** | Navigate |
| **A** | Toggle cheat |
| **B** | Back / close |
| **Y** | Show / hide active cheats list |
